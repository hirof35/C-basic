C++ Logic & String Manipulation Library本プロジェクトは、数理計算アルゴリズム、再帰構造、および標準ライブラリに依存しない独自の文字列操作関数（C-Style Strings）を実装した総合技術ライブラリです。技術的特徴1. 数理計算・再帰アルゴリズム (Mathematical Algorithms)反復構造（Iterative）と再帰構造（Recursive）の両面から、古典的アルゴリズムを実装しています。最大公約数 (GCD) / 最小公倍数 (LCM): ユークリッドの互除法を用いた実装。フィボナッチ数列: 動的計画法の基礎となる反復処理と再帰処理。ハノイの塔 (Tower of Hanoi): 再帰による解法と、ビット演算を用いた非再帰的な解法の両方を網羅。素数判定: $O(\sqrt{n})$ への最適化を意識した判定ロジック。ビット演算: 数値を16進数から2進数ビット列へと展開するデバッグ・表示用関数。2. 独自実装の文字列操作 (Custom String Library)string.h 等の標準ライブラリに頼らず、ポインタ演算のみで文字列操作の核となる機能を再構築しています。メモリ管理・コピー: strcpy, strncpy, strdup, strjoin連結・分割: strcat, split（デリミタによる動的分割）検索・比較: strstr, strcmp, strncmp書式変換: atoi（数値変換）、atoi_base（任意進数から10進数への変換）ケース変換: strupcase, strlowcase, strcapitalize（先頭大文字化）3. インタラクティブ・ユーティリティ数当てゲーム: random_device と mt19937 (メルセンヌ・ツイスタ) を使用した、現代的で高品質な乱数生成によるインタラクティブプログラム。セットアップと実行推奨環境C++11 以上（random_device および mt19937 使用のため）Microsoft Visual Studio (MSVC) または GCCビルド標準的なC++コンパイラでビルド可能です。各関数は独立性が高く設計されているため、必要なロジックのみを抽出して利用することも可能です。Bashg++ -std=c++11 main.cpp -o logic_tool
システム構造とポリシーポインタの活用: 配列インデックスではなくポインタのインクリメント（*s++）を多用し、実行速度とメモリ効率を重視した実装。構造的理解: 再帰とループの対比を通じて、計算量のトレードオフを視覚化。ライセンスMIT License
